## 1. 一句话总结

论文针对 **空中机械臂末端在复合扰动下的高精度控制**，提出 **「关节速度规划器（抵消基座浮动）+ RBF 神经网络 + 非线性扰动观测器（NDO）」** 的分层复合控制方案，并在 **5-DOF 臂 + 四旋翼** 真机平台上完成五组实验验证。

---

## 2. 问题背景与动机

### 2.1 要解决什么

空中机械臂（UAV + 多连杆臂）要实现抓取、检测等 **主动操作**，必须保证 **末端执行器（EE）跟踪精度**。与固定基座机械臂不同，存在三类复合扰动：

| 扰动类型 | 作用环节 | 机理 |
|----------|----------|------|
| **基座浮动（base-floating）** | 运动学环 | UAV 平移/姿态波动 → EE 位姿误差被 **放大** |
| **强耦合（coupling）** | 动力学环 | 臂运动对 UAV 施加反力/力矩 → 进一步加剧基座晃动 |
| **模型不确定性** | 动力学环 | 5-DOF 多关节系统惯性/科氏/重力参数难以精确辨识 |

### 2.2 与现有工作的差距（作者观点）

- 多数方法 **隐式** 通过改善 **飞控性能** 来间接提高 EE 精度，未 **系统处理** 作用在机械臂上的复合扰动。
- 已有 NN/DO 方法多只考虑 **动力学扰动**，忽略 **运动学环上的基座浮动**。
- 纯 IK/补偿类方法（如 [26][27][35]）停留在运动学层，**闭环稳定性** 缺乏严格保证。

### 2.3 与 `ee-stabilization` 的对应关系

| 维度 | 本文 | 本项目 |
|------|------|--------|
| 任务 | 跟踪 **时变轨迹**（圆轨迹等） | **世界系定点锁定** `T_E*` |
| 基座处理 | 解耦：UAV PID + 臂复合控制 | **仅臂补偿**，`T_B` 外生 |
| 运动学层 | 关节速度规划器 Eq.(21) | CLIK + `T_D = T_B^{-1} T_E*` |
| 动力学层 | NN + NDO | 模式 B: CTC；模式 C: OSC |
| 臂 DOF | 5-DOF（冗余） | 6-DOF（非冗余） |

**核心共鸣：** 二者都强调 **用臂的关节运动主动抵消基座浮动**，而非只依赖飞控。

## 3. 系统建模

### 3.1 坐标系

- **I（惯性系）**：本文记为 `P`，与项目 `{W}` 对应。
- **b（UAV 机体系）**：与项目 `{B}` / `base_link` 对应。
- **e（末端系）**：与项目 `{E}` / `link6` 对应。

### 3.2 运动学（Section II-A）

正运动学（Eq. 1）：

```text
P_e = P_b + R_b (P_1 + P_e^b)
R_e = R_b R_e^b
```

对时间求导并整理，得 EE 广义速度（Eq. 4–7）：

```text
v_e = J_b v_b + J_e q̇
ẋ_e = Φ_e^{-1} J_b Φ_b ẋ_b + Φ_e^{-1} J_e q̇
     = T_1 ẋ_b + T_2 q̇
```

**关键分解（Eq. 9）：**

```text
ẋ_e = d + T_2 q̇
```

其中 `d = T_1 ẋ_b` 为 **不可控的基座浮动扰动**，`|d| ≤ b_1`（有界）；`T_2 q̇` 为 **臂可控部分**。

> **与项目对照：** 项目通过 `T_D = T_B^{-1} T_E*` 在基座系生成时变期望，等价于让 `q` 产生 `-T_2^{-1} d` 方向的补偿；本文则在 **速度层** 显式构造 `q̇_re` 来抵消 `d`。

### 3.3 误差放大（Section II-C）

作者给出定量例子：若 UAV 位置/姿态允许误差为 **3 cm / 3°**，则 EE 在 x/y/z 方向的误差范围可放大至约 **±4.4 / ±6.2 / ±5.0 cm**。说明 **小幅基座扰动 → 大幅末端偏差**，与项目 limit_test 中「mount 扰动下 EE 漂移」现象一致。

### 3.4 动力学（解耦策略）

- **UAV**：标准四旋翼平移 PID + 姿态 PD（Section III-A），100 Hz 位置环 / 500 Hz 姿态环。
- **机械臂**（Eq. 12）：

```text
M_e(q) q̈ + C_e(q,q̇) q̇ + g_e(q) + τ_m(q,q̇,q̈) = τ
```

`τ_m` 为 **耦合扰动**（含臂-UAV 相互作用），与模型不确定性一并视为 **未知项**，由 NN 逼近。

> **与项目差异：** 项目用 Pinocchio 的 `M, h` 已知模型做 CTC/OSC；本文 **不假设已知动力学**，用 RBF-NN 在线估计。

## 4. 控制架构（Section III）

```text
┌─────────────────────────────────────────────────────────────┐
│  UAV 层（独立）                                               │
│  平移 PID (Eq.14) + 姿态 PD (Eq.18) + 微分平坦期望姿态        │
└─────────────────────────────────────────────────────────────┘
                              │
                              │ ẋ_b → d = T_1 ẋ_b
                              ▼
┌─────────────────────────────────────────────────────────────┐
│  机械臂 — 运动学层：关节速度规划器 (Eq.21)                     │
│  q̇_re = T_2^† ẋ_d − α T_2^T ε_1 − k_d T_2^† sat(aΔx)       │
└─────────────────────────────────────────────────────────────┘
                              │ q̇_re
                              ▼
┌─────────────────────────────────────────────────────────────┐
│  机械臂 — 动力学层：复合控制器 (Eq.27)                         │
│  τ = −K_s T_2^T ε_1 − K_v Δq̇ − K_a Δq̈ + Ŵ ϑ(ζ) + Ê_d      │
│  + NN 权值更新 (Eq.28) + NDO (Eq.29)                          │
└─────────────────────────────────────────────────────────────┘
```

控制频率：**机械臂复合控制器 1000 Hz**（Manifold 2C）；UAV 100/500 Hz。

## 5. 核心算法详解

### 5.1 关节速度规划器（贡献 1）

**任务误差：** `Δx = x_e − x_d`（惯性系下 EE 位置误差）

**势能函数（Eq. 19–20）：** 带饱和边界 `σ` 的分段势能，梯度 `ε_1 = ∂Φ/∂Δx`，使误差收敛到 **小邻域** 而非渐近零（实用上的「精确足够」）。

**参考关节速度（Eq. 21）：**

```text
q̇_re = T_2^† ẋ_d − α T_2^T ε_1 − k_d T_2^† sat(aΔx)
         ─────────   ─────────────   ─────────────────
         前馈跟踪      任务反馈         抗基座浮动项
```

- 第三项 `k_d T_2^† sat(aΔx)` 专门 **对抗基座浮动扰动 `d`**。
- 对比基线 Eq.(44)：`q̇_re = T_2^† ẋ_d − α T_2^T Δx`（无 `sat` 抗扰项）。

**误差桥接（Eq. 22–23）：**

```text
Δẋ = T_2 Δq̇ − α T_2 T_2^T ε_1 − k_d sat(aΔx) + d
```

当 `k_d > b_1`（扰动上界）时，可证 `Δx, Δq̇` 等 **一致最终有界（UUB）**。

### 5.2 复合动态控制器（贡献 2）

**RBF 神经网络（Eq. 24–26）：**

- 输入 `ζ = [q, q̇, q̈]^T`（`q̈` 由二阶滤波器获得）。
- 每个输出分量用 **51 神经元** 的 RBF 网络，共 4 维力矩输出（5-DOF 臂，论文式 (12) 中 τ ∈ R⁴，与具体驱动配置有关）。
- 逼近：`Υ(ζ) = W ϑ(ζ) + E_d`。

**控制律（Eq. 27）：**

```text
τ = −K_s T_2^T ε_1 − K_v Δq̇ − K_a Δq̈ + Ŵ ϑ(ζ) + Ê_d
```

**权值更新（Eq. 28）：** `Ẇ = −Δq̇ ϑ^T − k_2 Ŵ`（自适应律）

**NDO（Eq. 29–31）：** 补偿 NN 残差 `E_d`，改善 **瞬态** 性能（Remark 3：NN 权值收敛慢，NDO 在过渡期起作用）。

### 5.3 稳定性（Theorem 1）

Lyapunov 函数（Eq. 32）：

```text
V = Φ(Δx) + ½ Δq̇^T K_a Δq̇ + ½||W̃||² + ½ Ê_d^T Ê_d
```

证明得到 `V̇ ≤ −δ_0 V + C_0`，故 `Δx, Δq̇, W̃, Ê_d` **UUB**。**不需求已知 `M, C, g`**。

## 6. 实验设置

### 6.1 硬件平台

| 组件 | 规格 |
|------|------|

| 定位 | OptiTrack + MPU9250 融合 |
| 机载计算 | UAV: STM32F4/F7；臂: Manifold 2C @ **1000 Hz** |

### 6.2 参考轨迹（Eq. 43）

惯性系下 **椭圆/圆轨迹**（非定点保持）：

```text
x_d(t) = x_d(0) + 0.08 sin(ωt) / √5        (x)
y_d(t) = y_d(0) + 0.08 sin(ωt) / √(5/4)   (y)
z_d(t) = z_d(0) + 0.08 cos(ωt)            (z)
```

### 6.3 评价指标（Eq. 42）

- **μ**：平均绝对误差 `mean(||x_e − x_d||)`
- **ς**：标准差

## 7. 五组实验与结果

| 测试 | 目的 | 对比方法 | 关键结果 |
|------|------|----------|----------|
| **Test I** | 基座浮动抑制 | Eq.(44) vs 本文 Eq.(21)；6-DOF 振动台模拟基座扰动 | μ 改善 **23–31%**（x/y/z）；无抗扰时轨迹偏移 ~**4 cm**，本文 < **2 cm** |
| **Test II** | 动力学层 | Baseline I (仅 NN) vs II (仅 NDO) vs 本文 | 仅 NN **发散**；仅 NDO 稳态差；本文 NN+NDO **最优** |

## 8. 优缺点评价

### 8.1 优点

1. **问题分解清晰**：运动学层专门处理 base-floating，动力学层处理耦合+不确定性，符合 CHADC（复合分层抗扰）思想。
2. **理论完整**：从 Eq.(21) 到 Theorem 1 给出 UUB 证明，优于纯 IK 启发式补偿。
3. **工程验证充分**：5 组真机实验 + 冗余 DOF 仿真对比，指标量化（μ, ς）。
4. **NN+NDO 互补**：解决「NN 瞬态慢 / NDO 稳态模型失配」的经典矛盾（Remark 2–3）。
5. **与项目高度相关**：关节速度规划器思想可直接对标本项目的 **CLIK + V_D 前馈**。

### 8.2 局限

1. **任务不同**：全文为 **轨迹跟踪**，非 **世界系定点锁定**；定点仅 Test III 子场景之一。
2. **仍依赖 UAV 飞控**：解耦策略下 UAV 层 PID 限制 `ẋ_b`；项目假设 **mount 完全外生**，不控 UAV。
3. **5-DOF 冗余 vs 6-DOF 非冗余**：本文利用冗余避免关节越限（Test 冗余性验证）；项目 6DOF 任务 6D，无零空间。
4. **NN 在线负担**：51×4 神经元 + 1000 Hz，对嵌入式平台（如本项目 MIT 电机链）可能过重。
5. **仅位置控制**：实验主要报告 **位置** 误差，姿态控制未作为核心指标（与项目 6D SE(3) 目标有差距）。
6. **基座状态可测**：依赖 OptiTrack 等 **高精度外定位**；项目真机 `base_source=tf` 时估计噪声/延迟未讨论。

## 9. 与 `ee-stabilization` 三模式的对照

| 本项目模式 | 本文对应模块 | 可借鉴点 |
|------------|--------------|----------|
| **A（纯 IK）** | 无动力学；类似 [35] 反馈 IK 基线 | Test I 证明 **仅 IK/速度规划不够** 时需动力学层 |
| **B（CTC）** | 部分类似 NN 内环的「已知模型跟踪」 | 本文强调 **任务空间误差** 驱动；CTC 关节空间失配（项目 limit_test 已验证） |
| **C（OSC）** | `−K_s T_2^T ε_1` 任务反馈 + 动力学前馈 | 结构最接近；可叠加 **NDO** 估计 `\tau_m` |
| **规划层 CLIK** | Eq.(21) 速度规划器 | 可将 `sat(aΔx)` 抗扰项并入 `V_D` 或 CLIK 右端 |

**项目实验 vs 本文量级：**

- 项目标称 pos RMS：A **0.28 mm**，C **0.36 mm**（仿真，定点）
- 本文圆轨迹 μ：**3–10 mm** 量级（真机+轨迹+复合扰动）

二者 **不可直接数值对比**（任务、平台、扰动幅值不同），但 **相对排序**（运动学 < 任务空间动力学）一致。

## 10. 可落地的改进建议（针对本项目）

### 10.1 短期（不改架构）

1. **在 CLIK 中增加饱和反馈项**：参考 Eq.(21) 第三项，对 `\|e_p\|` 大时加强 `\dot{q}` 前馈。
2. **模式 C 叠加简化 NDO**：仅估计 `\tau_{dist}` 一阶，不引入完整 RBF-NN。
3. **复现 Test I 工况**：用项目 `BaseDisturbanceGenerator` + 振动台等价扰动，对比 CLIK vs CLIK+sat。

### 10.2 中期

1. **实现 Wang 式速度规划器** 作为模式 D，与 A/B/C 并列 benchmark。
2. **加速度级 CLIK**（Ruggiero 2025 P3）与本文 Eq.(21) 在 **高频 mount** 下 A/B 测试。
3. **分离评价**：位置 / 姿态 / 联合 `\|e_x\|` 报告，对齐本文 μ/ς 协议。

### 10.3 长期

1. 若引入 **冗余 DOF 或 7 轴**，采用 GJM + 零空间最小化反力矩（P4, D1）。
2. 若需 **UAV 联合控制**，迁移 Test III 对比的 [24] QP 框架。

## 13. 精读结论

本文是 **「仅通过机械臂抵消基座浮动、实现 EE 高精度操作」** 方向的代表性 TASE 工作。其 **关节速度规划器** 与 **NN+NDO 复合动态控制** 的分层思路，与 `ee-stabilization` 的 **「规划 CLIK + 控制 OSC」** 架构 **同构**。

对本项目最有价值的 takeaway：

1. **运动学层必须有显式 anti-base-floating 机制**（不只是 `T_D` 变换，还需速度层抗扰项）。
2. **任务空间反馈 + 扰动观测** 优于纯关节空间 CTC。
3. **真机指标 μ ~ mm–cm 级** 可作为对标参考，但需注意本文是 **轨迹跟踪 + 5-DOF 真机**，需按项目 **定点 6D** 任务换算。

**建议优先级：** 实现 Eq.(21) 第三项 → 模式 C 加 NDO → 与 limit_test 协议对齐复现实验。
