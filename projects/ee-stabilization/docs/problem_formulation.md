# Problem Formulation

**主题：** 浮动基座机械臂的末端世界系位姿稳定  
**文档性质：** 问题形式化——定义被控对象、输入输出、目标与约束。**先于**任何具体系统架构、控制律或软件实现。

---

## 1. 应用背景

无人机–机械臂等 **浮动基座（floating-base）** 系统中，机械臂基座 `{B}` 随载体发生未知、时变的平移与姿态运动。若关节不做主动补偿，末端执行器 `{E}` 在世界惯性系 `{W}` 中的位姿将随基座漂移，无法完成「对世界固定」的抓取、观测或操作。

**待解问题：** 在基座持续运动的情况下，通过关节控制，使末端在 `{W}` 中保持某一给定目标位姿。

---

## 2. 被控对象与坐标系

### 2.1 坐标系

| 符号 | 含义 |
|------|------|
| `{W}` | 世界惯性系，固定 |
| `{B}` | 基座系，固连机械臂安装点，随载体运动 |
| `{E}` | 末端系，固连末端执行器 |

系统由 **时变基座** 与 **固定于基座上的 n-DOF 串联机械臂**（本文中 n = 6）组成。基座位姿 `T_B(t) ∈ SE(3)` 由载体运动决定；关节向量 `q(t) ∈ R^n` 为可控配置。

### 2.2 位姿空间说明（SE(3) 与 R⁶）

刚体位姿 **`T = (R, p) ∈ SE(3)`**：`R ∈ SO(3)`，`p ∈ R³`。  
Lie 群 **SE(n)** 中的 **n 为所在欧氏空间维数**，不是自由度个数：

| 空间 | 含义 |
|------|------|
| **SE(3)** | 三维空间中的刚体变换；6 个自由度（3 平移 + 3 旋转） |
| **R⁶** | 六维向量，如 twist、任务误差（**不是**位姿群） |
| **SE(6)** | 六维空间中的刚体变换（21 维流形）——**与本问题无关** |

本问题中所有 **位姿**（`T_B`, `T_E`, `T_E*`）均在 **SE(3)** 上；**twist** 与 **任务误差** 在 **R⁶** 上。

---

## 3. 输入与输出

从控制问题角度，区分 **外生输入**、**反馈**、**控制输入（决策变量）** 与 **被控输出（评价量）**。

```text
  外扰 T_B(t)          任务设定 T_E*（常值）
        │                      │
        ▼                      ▼
  反馈 (q, q̇, T_B) ──→ [ 控制器 u = π(·) ] ──→ 控制输入 u
                              │
                              ▼
                        被控输出 T_E, e_x
```

### 3.1 输入（Inputs）

| 类别 | 符号 | 空间 | 说明 |
|------|------|------|------|
| **外扰** | `T_B(t)`，基座 twist | SE(3)， R⁶ | 载体引起的基座运动；**不可控** |
| **任务设定** | `T_E*` | SE(3) | 期望保持的末端世界系位姿；在 `t₀` 确定后**为常值** |
| **状态反馈** | `q`, `q̇` | R^n | 关节角与角速度 |
| **基座反馈** | `T_B`（及可选 twist） | SE(3)， R⁶ | 测量或估计的基座位姿/速度 |
| **模型** | `M, h, J`, 正运动学 | — | 机械臂已知动力学与运动学（见 §4） |

外扰 `T_B(t)` 对控制器是 **外生时变输入**；`T_E*` 是 **边界/设定条件**，不是每周期外部注入的信号。

### 3.2 输出（Outputs）

| 类别 | 符号 | 空间 | 说明 |
|------|------|------|------|
| **控制输入** | `u(t)` | 依执行器而定 | **待设计的控制律输出**；通常为关节力矩 `τ ∈ R^n`，或含位置/速度指令的执行器接口 |
| **被控输出** | `T_E(t)` | SE(3) | 实际末端位姿（由 `q` 与 `T_B` 经运动学确定） |
| **任务误差** | `e_x(t)` | R⁶ | 相对期望位姿的六维误差（见 §6） |
| **标量评价量** | `\|p_E - p_E*\|`, `\|e_R\|` | R | 世界系位置误差、姿态误差角 |

**小结：**  
- **输入** = 外扰 `T_B` + 反馈 `(q, q̇, T_B)` + 常值目标 `T_E*` + 机器人模型；  
- **待求输出** = 控制律 `u(t)`；  
- **评价输出** = `T_E` 及由此导出的 `e_x`。

## 4. 运动学与动力学模型

### 4.1 运动学

正运动学将关节配置映射为基座系下的末端位姿：

```text
T_E^B = f(q) ∈ SE(3)
```

世界系下：

```text
T_E = T_B · f(q)
```

末端 twist 与关节速度：

```text
ẋ = J(q) q̇,    ẋ ∈ R⁶,   J(q) ∈ R^{6×n}
```

### 4.2 动力学

机械臂在基座系下服从标准 n 自由度刚体动力学：

```text
M(q) q̈ + h(q, q̇) = τ
```

- `M(q)`：惯性矩阵  
- `h(q, q̇)`：重力、科氏/离心项  
- `τ ∈ R^n`：关节驱动力矩  

基座运动通过 `{B}` 的非惯性效应耦合进 `h` 及边界条件；若将臂视为固连于 `{B}`，则上式在 `{B}` 中成立，而 **`T_B(t)` 作为外参进入运动学**（§4.1）。

### 4.3 世界系锁定与基座系期望

任务要求末端在 `{W}` 中保持 `T_E*`。等价地，在 `{B}` 中末端应跟踪 **时变期望**：

```text
T_E* = const                          （t ≥ t₀）

T_D(t) = T_B(t)^{-1} T_E* ∈ SE(3)     （基座系期望末端位姿）
```

`T_D` 由几何关系 **唯一确定**，不是独立的设计自由度：基座动，则 `{B}` 中为保持 `{W}` 不动，末端必须做反向补偿。

---

## 5. 扰动与可观测性

### 5.1 基座扰动

基座轨迹 `T_B(t)` 视为 **有界、时变、不可控** 的外扰，可能包含：

- 平移：位置在某一空间包络内变化；  
- 姿态：绕各轴有界角位移；  
- 频谱：从缓变到较高频率的载体晃动。

控制器 **不能** 改变 `T_B`，只能通过 `τ`（或等效执行器输入）改变 `q`，进而改变 `T_E`。

### 5.2 可观测性假设

闭环控制假定可在线获得：

- `q`, `q̇`（关节编码器或等价测量）；  
- `T_B`（载体系估计、外部定位或仿真真值）。

未建模项（传感器噪声、模型误差、执行器带宽限制等）视为 **集总不确定性**，不展开为扰动模型的一部分，但影响可实现的误差下界。

---

## 6. 控制目标

### 6.1 主目标

使实际末端世界系位姿跟踪锁定目标：

```text
T_E(t) → T_E*     ⇔     f(q) → T_D(t)   （在 {B} 中）
```

在持续基座扰动下，一般要求 **有界跟踪**（误差有界且尽可能小），而非渐近收敛到零（因 `T_D` 时变且存在约束）。

### 6.2 任务误差

在基座系中，设 `T_D = (R_D, p_D)`，`T_E^B = (R_E, p_E)`：

```text
e_p = p_D - p_E
e_R = Log(R_E^T R_D)              （SO(3) 轴角向量）
e_x = [e_p; e_R] ∈ R⁶
```

世界系等价评价：

```text
位置误差：  ||p_E^W - p_E*||
姿态误差：  ||Log((R_E^W)^T R_E*)||
```

其中 `p_E^W`, `R_E^W` 为 `T_E` 在世界系下的分量。

### 6.3 优化型表述（可选）

在约束下，可形式化为：求因果控制律 `u(t) = π(·)`，使在扰动包络 `T_B ∈ 𝒯` 内最小化任务误差，例如：

```text
min_π   sup_{T_B ∈ 𝒯}  ∫_{t₀}^{t₀+T}  ρ(e_x(t)) dt
s.t.    M(q)q̈ + h(q,q̇) = τ(u, q, q̇)
        q ∈ Q,  τ ∈ T,  u = π(q, q̇, T_B, T_E*, t)
```

`ρ` 为非负代价（如 `||e_x||²`）；`Q`, `T` 为关节与力矩可行集。  
Problem formulation **不指定** `π` 的具体形式（关节空间、任务空间、或分层结构均属后续 Method）。

---

## 7. 约束

| 类型 | 形式 | 说明 |
|------|------|------|
| 关节位置 | `q_min ≤ q ≤ q_max` | 机械限位 |
| 关节速度 | `\|q̇_i\| ≤ q̇_max,i` | 速度与安全 |
| 关节力矩 | `\|τ_i\| ≤ τ_max,i` | 执行器饱和 |
| 奇异/位形 | 工作空间内避奇异（软或硬） | 保证 `J` 可逆性或数值稳定 |

所有约束在问题中视为 **硬约束** 或 **可行域边界**；具体数值由平台给定，不属于 Problem Formulation 本身。

---

## 8. 假设

1. **刚体：** 连杆与基座为刚体；弹性、间隙、迟滞忽略。  
2. **已知模型：** `M, h, J, f` 已知（允许有界参数误差，但不改变问题结构）。  
3. **全关节反馈：** `q`, `q̇` 可测。  
4. **基座可测：** `T_B`（及所需速度）可测或估计。  
5. **目标固定：** `T_E*` 在 `t₀` 确定后不变。  
6. **无环境接触：** 末端不与外界产生未建模接触力。  
7. **单臂：** 不考虑多臂协调或载体–臂耦合控制（载体仅通过 `T_B` 进入问题）。

---

## 9. 形式化问题陈述

**给定：**

| 类型 | 内容 |
|------|------|
| 机械臂 | n-DOF 串联臂，运动学 `f`、动力学 `(M, h)`、约束集 `(Q, T)` |
| 初始条件 | `q(t₀), q̇(t₀)` |
| 任务边界 | `T_E* ∈ SE(3)`（`t₀` 时刻给定的世界系目标位姿） |
| 外扰类 | `T_B : [t₀, ∞) → SE(3)`，属于某一有界扰动集 `𝒯` |
| 反馈 | 时刻 `t` 可测 `(q, q̇, T_B)` |

**求：** 因果控制律

```text
u(t) = π(q, q̇, T_B, T_E*, t)
```

使得对 `∀ T_B(·) ∈ 𝒯`：

1. **任务跟踪：** `e_x(t)` 有界，且 `\|e_x\|` 在适当意义下尽可能小；  
2. **约束满足：** `q(t) ∈ Q`，`u(t)` 诱导的 `τ(t) ∈ T`；  
3. **因果性：** `u(t)` 仅依赖 `t` 及当前与过去测量，不依赖未来 `T_B`。

**等价一句话：**

> 在时变基座 `{B}` 上，设计反馈控制，使末端 `{E}` 在世界系 `{W}` 中保持固定位姿 `T_E*`，并满足关节与执行器约束。

---

## 10. 性能评价（问题层面）

Problem formulation 只规定 **评价什么**，不规定具体阈值或实验协议。

| 指标 | 定义 |
|------|------|
| 位置误差 | `\|p_E^W(t) - p_E*\|` |
| 姿态误差 | `\|Log((R_E^W)^T R_E*)\|` |
| 任务误差范数 | `\|e_x(t)\|` |
| 时间域统计 | 在区间 `[t₀, t₀+T]` 上对误差的 mean、max、RMS、分位数等 |

**失效** 可定义为：在规定的扰动集 `𝒯` 与记录时长 `T` 下，误差统计量超过任务要求的容限（容限由应用给定，不属于本数学问题定义）。

---

## 11. 问题范围

**包含：**

- 浮动基座 + 固定 DOF 机械臂的 **世界系位姿保持**；  
- 由 `T_E*` 与 `T_B(t)` 导出的 **时变基座系期望** `T_D(t)`；  
- 任务空间误差 `e_x` 作为 **主评价量**；  
- 动力学与执行器约束下的 **反馈控制律设计问题**。

**不包含：**

- 无人机/载体本体控制（`T_B` 为外给）；  
- 末端轨迹跟踪（时变 `T_E*(t)`）；  
- 抓取力控、环境接触、避障路径规划；  
- 具体控制律（IK、CTC、OSC 等）、软件架构、通信接口或实验实现。
